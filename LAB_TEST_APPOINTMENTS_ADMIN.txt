LAB TEST APPOINTMENTS - ADMIN MANAGEMENT
========================================

CHANGES MADE:

1. BACKEND - labTestController.js
   - Added getAllLabTestAppointments() - Get all lab test appointments for admin
   - Added updateLabTestAppointment() - Upload test result URL and mark as COMPLETED
   - Added deleteLabTestAppointment() - Delete lab test appointment
   - Updated getPatientLabTests() - Status changes to COMPLETED when TEST_FILE_URL exists

2. BACKEND - adminRoutes.js
   - Added GET /api/admin/lab-test-appointments
   - Added PUT /api/admin/lab-test-appointments/:id
   - Added DELETE /api/admin/lab-test-appointments/:id

3. FRONTEND - AdminDashboard.jsx
   - Changed "Doctors" button to "Lab Test Appointments" with 🧪 icon
   - Added lab-test-appointments to all endpoint mappings
   - Added state: showEditModal, editingAppointment, testFileUrl
   - Added table rendering for lab test appointments showing:
     * Token, Patient (name + email), Test Name, Department
     * Technician, Price, Status (PENDING/COMPLETED)
     * Upload Result / View buttons
   - Added EditLabTestModal component for uploading test result URL
   - Status badge: Yellow for PENDING, Green for COMPLETED

WORKFLOW:
1. Admin clicks "Lab Test Appointments" in sidebar
2. Sees all lab test appointments with status
3. Clicks "Upload Result" on PENDING appointments
4. Enters test file URL (e.g., https://example.com/result.pdf)
5. Clicks "Upload & Complete"
6. Status changes to COMPLETED
7. Patient sees COMPLETED status in their dashboard
